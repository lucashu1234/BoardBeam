# BoardBeam

BoardBeam 是一个面向 Windows 10 / Windows 11 的开源课堂演示工具，适合教师、培训讲师和技术分享者在投屏或授课时进行屏幕批注、截图贴图、聚光灯强调、倒计时、GIF 录屏和滚动长截图。

项目目标是提供一个轻量、可离线使用、可二次开发的 ZoomIt / PixPin 风格工具。

## 功能特性

- 全局快捷键：一键进入批注、缩放、截图、录屏、聚光灯等模式。
- 屏幕批注：画笔、荧光笔、直线、箭头、矩形、椭圆、文字、编号标记、答案遮罩、模糊笔。
- 演示辅助：冻结缩放、白板、黑板、课堂倒计时、聚光灯/激光笔。
- 截图工具：区域复制、区域保存、延时截图、窗口截图、截图历史。
- 贴图工具：区域贴图、最近截图贴图、窗口贴图，支持缩放、透明度、锁定移动。
- 取色工具：截图模式下可复制鼠标位置颜色。
- GIF 录屏：拖选区域录制 GIF，支持快捷键停止。
- 滚动长截图：拖选滚动区域后自动滚动、识别重叠并拼接保存。
- 可配置快捷键：托盘菜单中可启用、禁用、修改和恢复默认快捷键。

## 系统要求

- Windows 10 或 Windows 11
- .NET Framework 4.x 运行环境
- 构建源码需要 Windows 自带的 .NET Framework C# 编译器：

```text
C:\Windows\Microsoft.NET\Framework64\v4.0.30319\csc.exe
```

当前项目不依赖 .NET SDK，也不依赖第三方 NuGet 包。

## 构建

在 PowerShell 中运行：

```powershell
.\build.ps1
```

构建产物：

```text
bin\BoardBeam.exe
```

## 运行

双击运行：

```text
bin\BoardBeam.exe
```

程序启动后会进入系统托盘。双击托盘图标默认进入批注模式，也可以右键托盘图标打开菜单。

## 默认快捷键

| 快捷键 | 功能 |
| --- | --- |
| `F9` | 一键打开/关闭批注画线 |
| `Ctrl + 1` | 冻结缩放 |
| `Ctrl + 2` | 批注画线 |
| `Ctrl + 3` | 课堂倒计时 |
| `Ctrl + 4` | LiveZoom 入口，当前为预留功能 |
| `Ctrl + Shift + 4` | LiveDraw 透明批注 |
| `Ctrl + 5` | 区域 GIF 录屏，录制中再次按下可停止 |
| `Ctrl + 6` | 拖选区域截图并复制到剪贴板 |
| `Ctrl + Shift + 6` | 拖选区域截图并保存文件 |
| `Ctrl + Alt + 6` | OCR 入口，当前为预留功能 |
| `Ctrl + 7` | DemoType 下一段文本 |
| `Ctrl + Shift + 7` | DemoType 上一段文本 |
| `Ctrl + Alt + T` | 文字输入 |
| `Ctrl + Alt + L` | 聚光灯/激光笔 |
| `Alt + 1` | PixPin 风格截图选择 |
| `Alt + 2` | 拖选区域并贴图 |
| `Alt + 3` | 3 秒延时后进入截图选择 |
| `Alt + 4` | 打开截图历史 |
| `Alt + 5` | 把鼠标下窗口贴到屏幕 |
| `Alt + Shift + 5` | 复制鼠标下窗口截图 |
| `Alt + 6` | 滚动长截图 |
| `Alt + Shift + 2` | 把最近截图或剪贴板图片贴到屏幕 |

再次按同一个全局快捷键会退出对应模式。快捷键可在托盘菜单“设置快捷键”中修改。

配置文件保存位置：

```text
%AppData%\BoardBeam\settings.ini
```

设置窗口会阻止重复快捷键，也会阻止单独字母/数字这类容易影响正常输入的全局快捷键；如需无组合键，建议使用 `F1`-`F24`。

鼠标操作：

- 鼠标右键按住：临时橡皮。
- 鼠标中键拖动或 `Alt + 左键拖动`：平移缩放画面。

- `Shift + 拖动`：直线锁角。
- `Ctrl + 拖动`：临时矩形。
- `Ctrl + Shift + 拖动`：临时箭头。

文字输入：

- `Enter`：换行。
- `Ctrl + Enter`：确认输入。
- `Esc`：取消输入。

## 截图与贴图

PixPin 风格截图模式：

- `Alt + 1`：进入截图选择模式。
- 鼠标拖选：选择区域。
- 悬停窗口：自动高亮，单击即可选中。
- `Space`：捕捉鼠标下窗口为选区。
- 方向键：移动选区。
- `Ctrl + 方向键`：调整选区大小。
- 按住 `Shift`：加快移动或调整步进。

选区完成后自动进入标注模式，可使用以下工具进行标注：

| 按键 | 工具 |
| --- | --- |
| `P` | 画笔 |
| `H` | 荧光笔 |
| `L` | 直线 |
| `A` | 箭头 |
| `R` | 矩形 |
| `O` | 椭圆 |
| `V` | 遮罩 |
| `X` | 马赛克 |
| `M` | 编号标记 |
| `E` | 橡皮 |
| `T` | 文字 |
| `1`-`8` | 切换颜色 |
| `+` / `-` | 调整线宽 |
| `Ctrl + Z` | 撤销 |
| `Ctrl + Y` | 重做 |
| `Shift + 拖动` | 直线锁角 |

标注模式鼠标操作：

- 左键拖动：绘制当前工具的标注。
- 右键：临时橡皮擦（擦除标注）。
- 拖拽选区边缘手柄：调整选区大小。
- 拖拽选区内部：移动选区。
- 点击工具栏按钮：切换工具或执行操作。

标注完成后的操作：

| 操作 | 方式 |
| --- | --- |
| 复制到剪贴板 | `Enter` / `Ctrl + C` / 工具栏"复制" |
| 保存为图片 | `S` / `Ctrl + S` / 工具栏"保存" |
| 贴到屏幕 | 工具栏"贴图" / 非标注状态下 `P` 或 `T` |
| 退出标注模式 | `Esc`（回到选区调整） |
| 放弃选区退出 | `Esc` → `Esc` |
| 关闭截图 | `Esc` → `Esc` → `Esc` / 工具栏"关闭" |

其他截图模式功能：

- `C`：复制鼠标处颜色，格式为 `#RRGGBB`。
- `Shift + C`：复制鼠标处颜色，格式为 `rgb(r, g, b)`。
- 鼠标右键（非标注时）：快速贴图并关闭。

贴图窗口：

- 左键拖动：移动贴图。
- 鼠标滚轮：缩放贴图。
- `Ctrl + 鼠标滚轮`：调整透明度。
- 右键菜单：复制、保存、设置缩放、设置透明度、锁定移动、关闭。
- `Esc`：关闭贴图。
- `Ctrl + C`：复制贴图。
- `Ctrl + S`：保存贴图。

截图默认保存到：

```text
%USERPROFILE%\Pictures\BoardBeam
```

## GIF 录屏

- `Ctrl + 5`：拖选区域后开始录制 GIF。
- 在录屏选区界面按 `Space`：选中鼠标下窗口，再按 `Enter` 开始录制。
- 录制中再次按 `Ctrl + 5`：停止并保存。
- 录屏控制窗“停止”：停止并保存。
- 录屏控制窗“打开”：定位到保存的 GIF 文件。

录屏默认保存到：

```text
%USERPROFILE%\Videos\BoardBeam
```

为避免 GIF 文件过大，单次录制最长 120 秒。更长时间、高清或带音频录制建议后续接入 MP4 录屏引擎。

## 滚动长截图

- `Alt + 6`：拖选滚动区域后自动滚动并拼接长图。
- 长截图默认保存到 `%USERPROFILE%\Pictures\BoardBeam`。
- 对普通网页、文档、列表区域效果较好。
- 复杂虚拟列表、懒加载内容或横向滚动区域可能需要分段截图。

## DemoType

DemoType 用于演示时自动输入预设文本。

- 如果 `bin` 目录下存在 `demotype.txt`，`Ctrl + 7` 会按片段输入文本。
- 多个片段之间用单独一行 `---` 分隔。
- 如果没有 `demotype.txt`，会使用当前剪贴板文本。

示例：

```text
第一段要演示输入的文字
---
第二段要演示输入的文字
```

## 当前限制

- GIF 录屏适合短片段演示，不适合长时间高清录制。
- MP4/带音频录屏尚未实现。
- OCR、真实 LiveZoom、Panorama recording、触控笔压感尚未实现。
- 滚动长截图基于滚轮滚动和图像重叠识别，复杂页面可能需要人工分段。
